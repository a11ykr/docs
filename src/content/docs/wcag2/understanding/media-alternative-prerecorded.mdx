### 의도

이 성공 기준의 목적은 시력이 너무 나빠서 [자막](../../../glossary/captions)을 제대로 읽을 수 없고, 청력이 너무 약하여 대화와 [오디오 해설](../../../glossary/audio-description)을 잘 들을 수 없는 사람에게 시청각 자료를 제공하는 것이다. 이를 위해 해당 비디오나 페이지와 동일한 인간 언어로 된 시간 기반 미디어 대체 수단을 제공한다.

이 접근 방식은 동기화된 미디어(시각 및 청각적 정보)의 모든 내용을 텍스트 형태로 제공하는 것이다. 시간 기반 미디어 대체 수단은 동기화된 미디어 콘텐츠에서 일어나는 모든 것을 연속적으로 기술하며, 책처럼 읽을 수 있다. 오디오 해설과 달리 비디오 일부에 대한 설명은 대화의 공백에만 제한되지 않고, 시각적 맥락, 배우의 행동·표정, 기타 시각적 자료를 모두 포함한다. 또한 웃음, 화면 밖 목소리 등 비언어적 소리도 설명하며, 모든 대사의 대본을 포함한다. 설명과 대사 대본의 순서는 실제 동기화 미디어의 진행 순서와 동일하다. 따라서 시간 기반 미디어 대체 수단은 오디오 해설만으로는 제공되지 않는 훨씬 더 완전한 표현을 제공한다.

동기화 미디어에 상호작용이 포함된 경우(예: '질문에 대답하려면 지금 누르세요'), 시간 기반 미디어 대체 수단은 하이퍼링크 등 동등한 기능을 제공해야 한다.

시력이 나빠 자막을 읽을 수 없으면서 청력이 약하여 대화를 들을 수 없는 사람은, 점자 디스플레이를 통해 시간 기반 미디어 대체 수단에 접근할 수 있다.

<Card title="참고">
1.2.3, 1.2.5, 1.2.8은 서로 겹치는 부분이 다소 있다. 이는 저작자에게 최소 준수 수준의 선택권을 주고 더 높은 수준에서는 추가적인 요구 사항을 제공하기 위한 것이다. 적합 기준 1.2.3의 레벨 A에서 저작자는 오디오 해설이나 텍스트를 전체 대체 수단으로 제공할 수 있다. AA 레벨을 준수하려는 경우 저작자는 성공 기준 1.2.5에 따라 오디오 해설을 제공해야 한다. 이는 1.2.3에 대한 대안을 선택한 경우 이미 충족된 요구 사항이고, 그렇지 않은 경우 추가 요구 사항이다. 성공 기준 1.2.8에 따라 레벨 AAA에서는 확장된 텍스트 설명을 제공해야 한다. 이는 오디오 해설만 제공하여 1.2.3과 1.2.5가 모두 충족되는 경우 추가 요구 사항이다. 그러나 텍스트 설명을 제공하여 1.2.3이 충족되고 오디오 설명에 대한 1.2.5 요구 사항이 충족된 경우 1.2.8에서는 새로운 요구 사항을 추가하지 않는다.
</Card>

### 이점

*   시청각 장애가 있어 화면이 잘 보이지 않거나 전혀 볼 수 없고, 소리를 거의 들을 수 없거나 전혀 들을 수 없는 사람도 시청각 자료의 정보를 얻을 수 있다.

### 예제

<dl>
    <dt>예시 1. 교육용 비디오에 대한 시간 기반 미디어 대체 수단</dt>
    <dd>
        한 커뮤니티 센터는 고객이 볼 교육 비디오를 구입하여 내부 인트라넷에 게시한다. 이 비디오에서는 새로운 기술 사용법을 설명하며, 한 사람이 말하면서 동시에 시연을 보여준다. 센터는 시연을 볼 수 없고 설명을 들을 수도 없는 클라이언트를 포함한 모든 사용자가 내용을 이해할 수 있도록 시간 기반 미디어 대체 수단을 제공한다.
    </dd>

### 기법

이 섹션의 각 번호가 매겨진 항목은 접근성 가이드라인 실무 그룹이 이 성공 기준을 충족하기에 충분하다고 판단하는 기법 또는 기법의 조합을 나타낸다. 기법은 기준의 최소 요구 사항을 넘어설 수 있다. 이러한 기법에서 포괄하지 않은 기준 충족의 다른 방법이 있을 수 있다. 다른 기법 사용에 대한 정보는 [WCAG 성공 기준에 대한 기법 이해](../understanding-techniques), 특히 "기타 기법" 섹션을 참조하라.

#### 충분 기법

아래에서 귀하의 콘텐츠와 일치하는 상황을 선택하라. 각 상황에는 해당 상황에 충분하다고 알려져 있고 문서화된 기법 또는 기법의 조합이 포함된다.

##### 상황 A: If the content is prerecorded synchronized media:

*   [G69: Providing an alternative for time based media](https://www.w3.org/WAI/WCAG22/Techniques/general/G69) using one of the following techniques
    *   [G58: Placing a link to the alternative for time-based media immediately next to the non-text content](https://www.w3.org/WAI/WCAG22/Techniques/general/G58)
*   Linking to the alternative for time-based media using one of the following techniques
    *   [H53: Using the body of the object element](https://www.w3.org/WAI/WCAG22/Techniques/html/H53)

##### 상황 B: If the content is prerecorded video-only:

*   [G159: Providing an alternative for time-based media for video-only content](https://www.w3.org/WAI/WCAG22/Techniques/general/G159)
