## 요약

<dl>
<dt>목표</dt>
<dd>사용자는 실수를 피할 수 있다.</dd>
<dt>할 일</dt>
<dd>현재 수행 중인 기능에 대해 사용자에게 도움을 제공한다.</dd>
<dt>중요성</dt>
<dd>인지 장애나 기타 장애가 있는 사람이 자신의 작업을 더 쉽게 완료할 수 있다.</dd>
</dl>

### 의도

이 성공 기준의 목적은 사용자가 실수를 방지하도록 돕는 것이다. 일부 장애가 있는 사용자는 장애가 없는 사용자보다 실수할 가능성이 더 높다. 상황에 맞는 도움말을 사용하여 사용자는 자신이 수행 중인 작업을 추적하지 않고도 작업을 수행하는 방법을 알아낼 수 있다.

상황에 맞는 도움말은 레이블이 모든 기능을 설명하기에 충분하지 않은 경우에만 제공되어야 한다. 상황에 맞는 도움말의 존재는 사용자에게 명확해야 하며 필요할 때마다 얻을 수 있어야 한다.

콘텐츠 저작자가 도움말 텍스트를 제공할 수도 있고, 사용자 에이전트가 기술별 프로그래밍 방식으로 결정된 정보를 기반으로 도움말 텍스트를 제공할 수도 있다.

### 이점

*   텍스트 입력 지원은 텍스트 입력이 필요한 양식이나 기타 장소에 텍스트를 쓰는 데 종종 어려움을 겪는 쓰기 장애가 있는 개인, 읽기 및 지적 장애가 있는 사람들에게 도움이 된다.
*   또한 이러한 종류의 지원은 노령화되어 텍스트 입력 및/또는 마우스 조작에 어려움을 겪는 사람들에게 도움이 된다.

### 예제

<dl>
    <dt>온라인 입사 지원</dt>
    <dd>
        일부 질문은 신규 구직자가 이해하기 어려울 수 있다. 각 질문 옆에 있는 도움말 링크는 각 질문에 대한 지침과 설명을 제공한다.
    </dd>
</dl>

### 기법

이 섹션의 각 번호가 매겨진 항목은 접근성 가이드라인 실무 그룹이 이 성공 기준을 충족하기에 충분하다고 판단하는 기법 또는 기법의 조합을 나타낸다. 기법은 기준의 최소 요구 사항을 넘어설 수 있다. 이러한 기법에서 포괄하지 않은 기준 충족의 다른 방법이 있을 수 있다. 다른 기법 사용에 대한 정보는 [WCAG 성공 기준에 대한 기법 이해](../understanding-techniques), 특히 "기타 기법" 섹션을 참조하라.

#### 충분 기법

아래에서 귀하의 콘텐츠와 일치하는 상황을 선택하라. 각 상황에는 해당 상황에 충분하다고 알려져 있고 문서화된 기법 또는 기법 조합이 포함된다.

##### 상황 A: If a form requires text input:

*   [G71: Providing a help link on every Web page](https://www.w3.org/WAI/WCAG22/Techniques/general/G71)
*   [G193: Providing help by an assistant in the Web page](https://www.w3.org/WAI/WCAG22/Techniques/general/G193)
*   [G194: Providing spell checking and suggestions for text input](https://www.w3.org/WAI/WCAG22/Techniques/general/G194)
*   [G184: Providing text instructions at the beginning of a form or set of fields that describes the necessary input](https://www.w3.org/WAI/WCAG22/Techniques/general/G184)

##### 상황 B: If a form requires text input in an expected data format:

*   [G89: Providing expected data format and example](https://www.w3.org/WAI/WCAG22/Techniques/general/G89)
*   [G184: Providing text instructions at the beginning of a form or set of fields that describes the necessary input](https://www.w3.org/WAI/WCAG22/Techniques/general/G184)

#### 조언 기법

준수를 위해 필수는 아니지만 콘텐츠에 더 쉽게 접근할 수 있도록 다음과 같은 추가 기법을 고려해야 한다. 모든 기법을 사용할 수 없거나 모든 상황에서 효과적인 것은 아니다.

*   [H89: Using the title attribute to provide context-sensitive help](https://www.w3.org/WAI/WCAG22/Techniques/html/H89)
