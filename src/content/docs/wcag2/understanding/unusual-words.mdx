(Level AAA)

<Card title="성공 기준(SC)">
[관용어](../../../glossary/idiom)와 [전문용어](../../../glossary/jargon)를 포함하여, [특이하거나 제한된 방식으로 사용된](../../../glossary/used-in-an-unusual-or-restricted-way) 단어나 문구의 정의를 식별할 수 있는 [매커니즘](../../../glossary/mechanism)을 제공해야 한다.
</Card>

## 요약

<dl>
<dt>목표</dt>
<dd>사용자는 특이한 단어의 의미를 식별하고 학습할 수 있다.</dd>
<dt>할 일</dt>
<dd>기술 전문 용어와 특이한 용어에 대한 정의를 제공한다.</dd>
<dt>중요성</dt>
<dd>더 많은 사람들, 특히 인지 장애가 있는 사람들이 콘텐츠의 의미를 이해할 수 있다.</dd>
</dl>

### 의도

특정 장애를 가진 사람들은 직접적이지 않은 단어 사용이나 전문용어 또는 특수한 사용법을 이해하기 어려울 수 있다. 특정 장애를 가진 사람들은 비유적 표현이나 전문적인 용법을 이해하기 어려울 수 있다. 이러한 청중에게 이런 매커니즘을 제공하는 것은 중요하다. 전문가가 아닌 일반 대중을 위한 전문 정보에서도 레벨 A 또는 레벨 AA 준수만을 주장하더라도 이 성공기준을 만족시키는 것이 권장된다.

### 이점

이 성공 기준은 다음과 같은 인지 장애, 언어 장애, 학습 장애가 있는 사람들에게 도움이 될 수 있다.

*   단어를 해독하는 데 어려움을 겪는
*   단어와 문구를 이해하는 것에 어려움을 겪는
*   이해를 돕기 위한 맥락을 사용하는 데 어려움을 겪는

또한 다음과 같은 시각 장애가 있는 사람들에게도 도움이 될 것이다.

*   화면 확대기를 사용하여 확대하면 맥락을 잃는

### 예제
<dl>
    <dt>특이한 방식으로 사용되는 단어에 대한 정의가 포함된 텍스트</dt>
    <dd>
        사전의 "계층"이나 기타 자료의 목록에서 의미를 검색하면 목표한 정의가 표시되고 "계층"의 다른 출처에서 가져온 정의가 표시되지 않도록 구성한다. ("계층"은 올바른 정의를 찾을 가능성이 가장 높은 순서대로 사전 및 기타 참조 자료를 나열한다. 이를 통해 정의를 검색할 때 따를 순서를 제어할 수 있다.)
    </dd>
    <dt>정의를 포함한 용어집</dt>
    <dd>
        WCAG 2.0은 "텍스트"라는 단어를 특정한 방식으로 사용한다. 따라서 WCAG 2.0에서 "텍스트"라는 단어가 사용되면 동일한 웹 페이지 내의 용어집에 제공된 "텍스트"의 정의와 연결된다.
    </dd>
    <dt>페이지 하단에 제공되는 단어의 구체적인 정의</dt>
    <dd>단어에서 해당 정의로의 내부 링크도 페이지 내에 제공된다.</dd>
</dl>

### 기법

이 섹션의 각 번호가 매겨진 항목은 접근성 지침 실무 그룹이 이 성공 기준을 충족하기에 충분하다고 판단하는 기법 또는 기법의 조합을 나타낸다. 기법은 기준의 최소 요구 사항을 넘어설 수 있다. 이러한 기법에서 포괄하지 않은 기준 충족의 다른 방법이 있을 수 있다. 다른 기법 사용에 대한 정보는 [WCAG 성공 기준에 대한 기법 이해](../understanding-techniques), 특히 "기타 기법" 섹션을 참조하라.

#### 충분 기법

아래에서 귀하의 콘텐츠와 일치하는 상황을 선택하라. 각 상황에는 해당 상황에 충분하다고 알려져 있고 문서화된 기법 또는 기법의 조합이 포함된다.

##### 상황 A: If the word or phrase has a unique meaning within the Web page:

*   [G101: Providing the definition of a word or phrase used in an unusual or restricted way](https://www.w3.org/WAI/WCAG22/Techniques/general/G101) for the first occurrence of the word or phrase in a Web page using one of the following techniques:
    *   [G55: Linking to definitions](https://www.w3.org/WAI/WCAG22/Techniques/general/G55)
        
        *   [H40: Using description lists](https://www.w3.org/WAI/WCAG22/Techniques/html/H40)
    *   [G112: Using inline definitions](https://www.w3.org/WAI/WCAG22/Techniques/general/G112)
        
        *   [H54: Using the dfn element to identify the defining instance of a word](https://www.w3.org/WAI/WCAG22/Techniques/html/H54)
*   [G101: Providing the definition of a word or phrase used in an unusual or restricted way](https://www.w3.org/WAI/WCAG22/Techniques/general/G101) for each occurrence of the word or phrase in a Web page using one of the following techniques:
    *   [G55: Linking to definitions](https://www.w3.org/WAI/WCAG22/Techniques/general/G55)
        
        *   [H40: Using description lists](https://www.w3.org/WAI/WCAG22/Techniques/html/H40)
    *   [G62: Providing a glossary](https://www.w3.org/WAI/WCAG22/Techniques/general/G62)
    *   [G70: Providing a function to search an online dictionary](https://www.w3.org/WAI/WCAG22/Techniques/general/G70)

##### 상황 B: If the word or phrase means different things within the same Web page:

*   [G101: Providing the definition of a word or phrase used in an unusual or restricted way](https://www.w3.org/WAI/WCAG22/Techniques/general/G101) for each occurrence of the word or phrase in a Web page using one of the following techniques:
    *   [G55: Linking to definitions](https://www.w3.org/WAI/WCAG22/Techniques/general/G55)
        
        *   [H40: Using description lists](https://www.w3.org/WAI/WCAG22/Techniques/html/H40)
    *   [G112: Using inline definitions](https://www.w3.org/WAI/WCAG22/Techniques/general/G112)
        
        *   [H54: Using the dfn element to identify the defining instance of a word](https://www.w3.org/WAI/WCAG22/Techniques/html/H54)
