<Aside title="참고 1">
이 지침에서 사용된 “웹 기술(<span lang="en">Web Technology</span>)”과 “기술(<span lang="en">technology</span>)”(독자적으로 사용될 때)이라는 단어는 모두 웹 콘텐츠 기술(<span lang="en">Web Content Technologies</span>)을 의미한다.
</Aside>

<Aside title="참고 2">
웹 콘텐츠 기술에는 정적 웹 페이지에서부터 동기화된 미디어 표현(<span lang="en">presentation</span>), 동적 웹 애플리케이션에 이르기까지 최종 사용자 경험을 조성하기 위해 개발자가 단독으로 또는 함께 사용할 수 있는 마크업 언어, 데이터 형식 또는 프로그래밍 언어가 포함될 수 있다.
</Aside>

<Card title="예제" icon="open-book">
웹 콘텐츠 기술의 몇 가지 일반적인 예로는 HTML, CSS, SVG, PNG, PDF, Flash, JavaScript가 있다.
</Card>
