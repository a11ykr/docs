실제 이벤트(<span lang="en">event</span>)에서 캡처하여 방송 지연(<span lang="en">broadcast delay</span>) 없이 수신기로 전송되는 정보


<Aside title="참고 1">
방송 지연은 예를 들어 방송사가 오디오(또는 비디오) 피드(<span lang="en">feed</span>)에 신호를 주거나 검열할 시간을 주기 위해 사용되지만, 상당한 편집을 하기에는 충분하지 않은, 아주 짧은(일반적으로 자동화된) 지연 시간을 말한다.
</Aside>

<Aside title="참고 2">
완전히 컴퓨터로 생성된 정보는 실시간이 아니다.
</Aside>
