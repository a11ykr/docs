다른 항목들과 대비하여 동일한 위치

<Aside title="참고">
다른 항목이 원래 순서에서 삽입되거나 제거되더라도, 항목은 상대 순서가 동일한것으로 간주한다. 예를 들어, 1차 메뉴를 확장하면 부가적인 상세 수준이 삽입되거나 2차 메뉴 섹션이 읽기 순서에 삽입될 수 있다.
</Aside>
