사용했을 때 동일한 결과가 도출되는

<Card title="예제" icon="open-book">
한 웹 페이지의 “검색(search)” 버튼과 다른 웹 페이지의 “찾기(find)” 버튼은 모두 검색어를 입력할 수 있는 양식(field)을 가지고 있고, 입력한 검색어와 관련된 주제를 웹사이트 내에 나열한다. 이 경우, 두 버튼은 동일한 기능을 가지고 있지만 레이블이 일관성 있게 부여되지 않은 것이다.
</Card>
