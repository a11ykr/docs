## [성공기준 이해 1.2.8](https://w3.org/TR/WCAG22#media-alternative-prerecorded):미디어 대체 수단 (사전 녹음/녹화된) (Level AAA)

<Card title="성공 기준 (SC)"></Card>

모든 [사전 녹음된](../../../glossary/prerecorded) [동기화된 미디어](../../../glossary/synchronized-media)와 사전 녹화된 [비디오 전용](../../../glossary/video-only) 미디어에는 [시간 기반 미디어에 대한 대체 수단](../../../glossary/alternative-for-time-based-media)을 제공해야 한다.

## 요약


목표

녹화된 영상을 더 많은 사람들이 이해할 수 있다.

할 일

동영상의 모든 콘텐츠에 해당하는 텍스트를 제공하라.

중요성

청각 장애인을 포함한 더 많은 사람들이 자신의 속도에 맞춰 콘텐츠를 더 잘 이해할 수 있다.

### 의도

이 성공 기준의 목적은 시력이 너무 나빠서 [자막](../../../glossary/captions)을 읽을 수 없고 청력이 너무 나빠서 대화와 [오디오 해설](../../../glossary/audio-description)을 확실히 들을 수 없는 개인이 시청각 자료를 사용할 수 있도록 하는 것이다. 이는 시간 기반 미디어에 대한 대체 수단을 제공함으로써 수행된다.

이 접근 방식에는 동기화된 미디어(시각 및 청각 모두)의 모든 정보를 텍스트 형식으로 제공하는 작업이 포함된다. 시간 기반 미디어의 대체 수단은 동기화된 미디어에서 표현되는 모든 것에 대한 설명을 제공한다. 시간 기반 미디어의 대체 수단은 책과 같은 것을 읽는 것이다. 오디오 해설과 달리 비디오 일부에 대한 설명은 기존 대화의 일시 정지에만 국한되지 않는다. 시각적 맥락, 배우의 행동과 표정, 기타 시각적 자료를 포함한 모든 시각적 정보에 대한 자세한 설명이 제공된다. 또한 말소리가 아닌 소리(웃음, 화면 밖의 목소리 등)도 설명되어 있으며, 모든 대화의 내용이 포함되어 있다. 설명 및 대화 내용의 순서는 동기화된 미디어 자체의 순서와 동일하다. 결과적으로 시간 기반 미디어의 대체 수단은 오디오 해설만 사용할 때보다 동기화된 미디어 콘텐츠를 훨씬 더 완벽하게 표현할 수 있다.

동기화된 미디어 프레젠테이션의 일부로 상호 작용이 있는 경우(예: "질문에 대답하려면 지금 누르세요") 시간 기반 미디어에 대한 대체 수단은 하이퍼링크 또는 병렬 기능을 제공하는 데 필요한 모든 것을 제공한다.

시력이 너무 나빠서 자막을 읽을 수 없고 청력이 너무 나빠서 대화를 확실히 들을 수 없는 사람들은 새로 고칠 수 있는 점자 디스플레이를 사용하여 시간 기반 미디어의 대체 수단에 접근할 수 있다.

참고

1.2.3, 1.2.5, 1.2.7의 경우 비디오 트랙의 모든 정보가 이미 오디오 트랙에 제공되어 있으면 오디오 해설이 필요없다.

1.2.3, 1.2.5, 1.2.8은 서로 겹치는 부분이 다소 있다. 이는 저작자에게 최소 준수 수준의 선택권을 주고 더 높은 수준에서는 추가적인 요구 사항을 제공하기 위한 것이다. 적합 기준 1.2.3의 레벨 A에서 저작자는 오디오 해설이나 텍스트를 전체 대체 수단으로 제공할 수 있다. AA 레벨을 준수하려는 경우 저작자는 성공 기준 1.2.5에 따라 오디오 해설을 제공해야 한다. 이는 1.2.3에 대한 대안을 선택한 경우 이미 충족된 요구 사항이고, 그렇지 않은 경우 추가 요구 사항이다. 성공 기준 1.2.8에 따라 레벨 AAA에서는 확장된 텍스트 설명을 제공해야 한다. 이는 오디오 해설만 제공하여 1.2.3과 1.2.5가 모두 충족되는 경우 추가 요구 사항이다. 그러나 텍스트 설명을 제공하여 1.2.3이 충족되고 오디오 설명에 대한 1.2.5 요구 사항이 충족된 경우 1.2.8에서는 새로운 요구 사항을 추가하지 않는다.

### 이점

*   청각 장애가 있는 사람, 잘 볼 수 없거나 전혀 볼 수 없는 사람, 잘 듣거나 전혀 들을 수 없는 사람은 시청각 프리젠테이션을 통해 정보에 접근할 수 있다.

### 예제

예제 1. 훈련 비디오를 위한 시간 기반 미디어의 대체 수단

커뮤니티 센터는 고객이 사용할 교육 비디오를 구입하여 센터 인트라넷에 게시한다. 영상에는 신기술의 사용법을 설명하는 내용이 담겨 있으며, 사람이 말하는 동시에 사물을 보여주는 모습도 담겨 있다. 커뮤니티 센터는 동기화된 미디어에서 시연을 볼 수 없거나 설명을 들을 수 없는 사람들을 포함하여 모든 클라이언트가 제공되는 내용을 더 잘 이해하는 데 사용할 수 있는 시간 기반 미디어에 대한 대체 수단을 제공한다.

#### 충분 기법

아래에서 귀하의 콘텐츠와 일치하는 상황을 선택하라. 각 상황에는 해당 상황에 충분하다고 알려져 있고 문서화된 기법 또는 기법의 조합이 포함된다.

##### 상황 A: If the content is prerecorded synchronized media:

*   [G69: Providing an alternative for time based media](https://www.w3.org/WAI/WCAG22/Techniques/general/G69) using one of the following techniques
    
    *   [G58: Placing a link to the alternative for time-based media immediately next to the non-text content](https://www.w3.org/WAI/WCAG22/Techniques/general/G58)
*   Linking to the alternative for time-based media using one of the following techniques
    
    *   [H53: Using the body of the object element](https://www.w3.org/WAI/WCAG22/Techniques/html/H53)

##### 상황 B: If the content is prerecorded video-only:

*   [G159: Providing an alternative for time-based media for video-only content](https://www.w3.org/WAI/WCAG22/Techniques/general/G159)
