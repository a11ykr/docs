### 의도

이 성공 기준의 목적은 시각 장애가 있는 사람들에게 동기화된 미디어의 시각적 정보에 대해 접근성을 제공하는 것이다. 이 성공 기준은 두 가지 접근 방식을 설명하며 둘 중 하나를 사용할 수 있다.

한 가지 접근 방식은 비디오 콘텐츠에 대한 오디오 해설(화면 해설)을 제공하는 것이다. 오디오 해설은 비디오의 일부를 사용할 수 없을 때 필요한 정보로 제공된 오디오 부분을 보강한다. 대화가 일시 중지된 동안 기본 사운드 트랙에서는 설명되거나 언급되지 않는 동작, 캐릭터, 장면 변경 및 화면 텍스트에 대한 중요한 정보를 오디오 해설이 제공한다.

두 번째 접근 방식은 동기화된 미디어의 모든 정보(시각 및 청각 모두)를 텍스트 형식으로 제공하는 것이다. 시간 기반 미디어의 대체 수단은 동기화된 미디어에서 진행되는 모든 것에 대한 실시간 설명을 제공한다. 시간 기반 미디어의 대체 수단이 시나리오나 책과 같은 것을 읽는 것이다. 오디오 해설과 달리 비디오 부분의 설명은 기존 대화의 일시 정지에만 국한되지 않는다. 시각적 맥락, 배우의 행동과 표정, 기타 시각적 자료를 포함한 모든 시각적 정보에 대한 자세한 설명이 제공된다. 또한 말소리가 아닌 소리(웃음, 화면 밖의 목소리 등)도 설명되어 있으며, 모든 대화의 내용이 포함되어 있다. 설명 및 대화 내용의 순서는 동기화된 미디어 자체의 순서와 동일하다. 결과적으로 시간 기반 미디어의 대체 수단이 오디오 해설만 사용할 때보다 동기화된 미디어 콘텐츠를 훨씬 더 완벽하게 표현할 수 있다.

동기화된 미디어에서 상호 작용하는 부분이 있다면(예: "질문에 대답하려면 지금 누르세요") 시간 기반 미디어에 대한 대체 수단은 하이퍼링크 또는 동일한 기능을 제공하는 데 필요한 모든 것을 제공한다.

<Card title="참고">

1.2.3, 1.2.5, 1.2.7의 경우 비디오 트랙의 모든 정보가 이미 오디오 트랙에 제공되어 있으면 오디오 설명이 필요없다.

1.2.3, 1.2.5, 1.2.8은 서로 겹치는 부분이 다소 있다. 이는 저작자에게 최소 준수 수준의 선택권을 주고 더 높은 수준에서는 추가적인 요구 사항을 제공하기 위한 것이다. 적합 기준 1.2.3의 레벨 A에서 저작자는 오디오 해설이나 텍스트를 전체 대체 수단으로 제공할 수 있다. AA 레벨을 준수하려는 경우 저작자는 성공 기준 1.2.5에 따라 오디오 해설을 제공해야 한다. 이는 1.2.3에 대한 대안을 선택한 경우 이미 충족된 요구 사항이고, 그렇지 않은 경우 추가 요구 사항이다. 성공 기준 1.2.8에 따라 레벨 AAA에서는 확장된 텍스트 설명을 제공해야 한다. 이는 오디오 해설만 제공하여 1.2.3과 1.2.5가 모두 충족되는 경우 추가 요구 사항이다. 그러나 텍스트 설명을 제공하여 1.2.3이 충족되고 오디오 설명에 대한 1.2.5 요구 사항이 충족된 경우 1.2.8에서는 새로운 요구 사항을 추가하지 않는다.

</Card>

### 이점

*   이 성공 기준은 움직이는 이미지를 인식하거나 이해하는 데 어려움이 있는 사람들을 포함하여 비디오 또는 기타 동기화된 미디어 콘텐츠를 시청하는 데 어려움을 겪는 사람들에게 도움이 될 수 있다.

### 기법

이 섹션에서 번호가 매겨진 각 항목은 WCAG 실무 그룹이 이 성공 기준을 충족하기에 충분하다고 간주하는 기법 또는 기법의 조합을 나타낸다. 그러나 이러한 특정 기법을 사용할 필요는 없다. 다른 기법 사용에 대한 자세한 내용은 [WCAG 성공 기준에 대한 기법 이해](understanding-techniques), 특히 "기타 기법" 섹션을 참고하라.

#### 충분 기법

*   [G69: Providing an alternative for time based media](https://www.w3.org/WAI/WCAG22/Techniques/general/G69) using one of the following techniques
    *   [G58: Placing a link to the alternative for time-based media immediately next to the non-text content](https://www.w3.org/WAI/WCAG22/Techniques/general/G58)
*   Linking to the alternative for time-based media using one of the following techniques
    *   [H53: Using the body of the object element](https://www.w3.org/WAI/WCAG22/Techniques/html/H53)
*   [G78: Providing a second, user-selectable, audio track that includes audio descriptions](https://www.w3.org/WAI/WCAG22/Techniques/general/G78)
*   [G173: Providing a version of a movie with audio descriptions](https://www.w3.org/WAI/WCAG22/Techniques/general/G173) using one of the following:
    *   [SM6: Providing audio description in SMIL 1.0](https://www.w3.org/WAI/WCAG22/Techniques/smil/SM6)
    *   [SM7: Providing audio description in SMIL 2.0](https://www.w3.org/WAI/WCAG22/Techniques/smil/SM7)
    *   Using any player that supports audio and video
*   [G8: Providing a movie with extended audio descriptions](https://www.w3.org/WAI/WCAG22/Techniques/general/G8) using one of the following:
    *   [SM1: Adding extended audio description in SMIL 1.0](https://www.w3.org/WAI/WCAG22/Techniques/smil/SM1)
    *   [SM2: Adding extended audio description in SMIL 2.0](https://www.w3.org/WAI/WCAG22/Techniques/smil/SM2)
    *   Using any player that supports audio and video
*   [G203: Using a static text alternative to describe a talking head video](https://www.w3.org/WAI/WCAG22/Techniques/general/G203)

준수를 위해 필수는 아니지만 콘텐츠에 더 쉽게 접근할 수 있도록 다음과 같은 추가 기법을 고려해야 한다. 모든 기법을 사용할 수 없거나 모든 상황에서 효과적인 것은 아니다.
