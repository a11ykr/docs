## 4. HTML 문서에서 ARIA 속성 사용에 대한 문서 준수 요구사항

이 섹션에서는 HTML 문서에서 ARIA 속성을 사용할 때 따라야 하는 구체적인 규칙과 요구사항을 설명한다. 이러한 요구사항은 웹 콘텐츠가 다양한 사용자 에이전트 및 보조 기술에서 일관되게 해석되고 접근 가능하도록 보장하는 데 중요하다.

주요 내용은 다음과 같다:

*   ARIA 속성을 사용하여 요소를 명명하는 요구사항 (4.1)
*   동등한 HTML 속성 대신 ARIA 속성을 사용하는 요구사항 (4.2)
*   사용 중단된(deprecated) ARIA 역할, 상태, 속성 및 HTML 속성에 대한 요구사항 (4.3)
*   ARIA 역할, 상태, 속성 값의 대소문자 요구사항 (4.4)
*   개별 HTML 요소별 ARIA 속성 사용 규칙 (이후 페이지에서 상세히 다룸)

<Aside type="note">
