## 2. HTML에서 벗어난 ARIA 의미 체계

ARIA는 HTML 요소의 의미 체계를 확장하여 접근성 API에 노출되는 방식을 변경할 수 있다. 저작자는 HTML 요소에 ARIA `role` 및 `aria-*` 속성을 추가할 수 있지만, HTML에서 ARIA 사용에 대한 저작자 요구사항을 준수해야 한다.

예를 들어, 다음은 `div` 요소를 사용하여 슬라이더 위젯을 만드는 방법이다.

<Card title="예제 1: ARIA를 사용하여 슬라이더 위젯 생성">
```html
<div role="slider" aria-label="Volume" aria-valuemin="0" aria-valuenow="40" aria-valuemax="100">
</div>
```
</Card>

이 예제에서 `div` 요소는 `role="slider"`를 통해 슬라이더 위젯으로 식별된다. `aria-label` 속성은 슬라이더에 접근 가능한 이름을 제공하고, `aria-valuemin`, `aria-valuenow`, `aria-valuemax` 속성은 슬라이더의 현재 값과 범위를 정의한다.
