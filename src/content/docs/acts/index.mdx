## 페이지 단위 점검 도구

*   [Accessibility Korea <Icon name="external" label="외부 링크" />](https://accessibility.kr/): 온라인 웹 접근성 진단 서비스
*   [Accessibility checker <Icon name="external" label="외부 링크" />](https://www.accessibilitychecker.org/): 영문. 일 3개 URL 무료 테스트 가능
*   [WAVE, Web Accessibility Evaluation Tool <Icon name="external" label="외부 링크" />](https://wave.webaim.org/)

## 개별 항목 검사도구

*   [Colour Contrast Analyser (CCA) <Icon name="external" label="외부 링크" />](https://developer.paciellogroup.com/color-contrast-checker/): 명도 대비 검사를 위한 설치용 소프트웨어. 윈도, 맥 지원.

## 브라우저 확장, 플러그인 외

*   [Lighthouse <Icon name="external" label="외부 링크" />](https://developer.chrome.com/docs/lighthouse/overview?hl=ko): 크롬 브라우저에 기본 탑재.
*   [axe DevTools <Icon name="external" label="외부 링크" />](https://www.deque.com/axe/): 브라우저 확장으로 설치 필요. 별도의 유료 버전 있음. [axe Accessibility Linter <Icon name="external" label="외부 링크" />](https://marketplace.visualstudio.com/items?itemName=deque-systems.vscode-axe-linter) 참고.
*   [OpenWAX <Icon name="external" label="외부 링크" />](https://goonoo.github.io/OpenWAX/): 크롬 확장 및 북마클릿 제공
*   [WAVE <Icon name="external" label="외부 링크" />](https://wave.webaim.org/extension/): 브라우저용 WAVE. 크롬, 파이어폭스, 엣지 확장 지원.
*   [Accessibility Insights <Icon name="external" label="외부 링크" />](https://accessibilityinsights.io/downloads/): 크롬, 엣지 확장 지원. 윈도용 소프트웨어 제공.
*   [kwcag a11y inspector <Icon name="external" label="외부 링크" />](https://chromewebstore.google.com/detail/kwcag-a11y-inspector/ngcmkfaolkgkjbddhjnhgoekgaamjibo): 타겟 사이즈 측정용 크롬 확장.
*   [WCAG Color contrast checker <Icon name="external" label="외부 링크" />](https://chromewebstore.google.com/detail/wcag-color-contrast-check/plnahcmalebffmaghcpcmpaciebdhgdf): 명도대비 측정용 크롬 확장

## 참고
*   [Web Accessibility Evaluation Tools List <Icon name="external" label="외부 링크" />](https://www.w3.org/WAI/test-evaluate/tools/list/): W3C에서 소개하는 접근성 점검 도구 목록
*   [크롬 접근성 기능 참조 <Icon name="external" label="외부 링크" />](https://developer.chrome.com/docs/devtools/accessibility/reference?hl=ko): 크롬 개발자 도구의 접근성 기능 소개
